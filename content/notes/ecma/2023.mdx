# Array 的非变异方法

```js
const arr = [3, 1, 4, 1, 5];

// 返回新数组，不修改原数组
arr.toReversed(); // [5, 1, 4, 1, 3]
arr.toSorted(); // [1, 1, 3, 4, 5]
arr.toSpliced(1, 2, "a", "b"); // [3, 'a', 'b', 1, 5]
arr.with(1, "replaced"); // [3, 'replaced', 4, 1, 5]

console.log(arr); // [3, 1, 4, 1, 5] 原数组未变
```

# Array 查找方法增强

```js
const arr = [1, 2, 3, 4, 5];

// 从后向前查找
arr.findLast((x) => x > 3); // 5
arr.findLastIndex((x) => x > 3); // 4
```
