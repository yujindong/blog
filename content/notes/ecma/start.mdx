# 解释

​​ECMA-262​​：​​JavaScript 语言的核心规范​​，定义了语法基础（如变量、数据类型、运算符、函数、类、模块等）、执行模型（作用域、闭包、事件循环）及内置对象（Array、String、Promise等）。​​目标​​：确保不同环境（如浏览器、Node.js）下 JavaScript 的行为一致性。

​​ECMA-402​​：​​国际化（Internationalization）规范​​，专注于​​区域敏感功能​​，如日期/时间格式、数字/货币格式化（Intl.DateTimeFormat、Intl.NumberFormat）、字符串比较（Intl.Collator）等。​​目标​​：使应用能适配不同语言、地区的本地化需求
