# 对象的 Rest/Spread 属性

```js
const obj = { a: 1, b: 2, c: 3, d: 4 };

// Rest 语法
const { a, b, ...rest } = obj;
console.log(rest); // { c: 3, d: 4 }

// Spread 语法
const newObj = { ...obj, e: 5 };
console.log(newObj); // { a: 1, b: 2, c: 3, d: 4, e: 5 }

// 对象克隆
const clone = { ...obj };
```

# 异步迭代 (for-await-of)

```js
async function* asyncGenerator() {
  yield Promise.resolve(1);
  yield Promise.resolve(2);
  yield Promise.resolve(3);
}

async function consumeAsync() {
  for await (const value of asyncGenerator()) {
    console.log(value); // 1, 2, 3
  }
}
```

# Promise.prototype.finally()

```js
fetch("/api/data")
  .then((response) => response.json())
  .catch((error) => console.error(error))
  .finally(() => {
    // 无论成功还是失败都会执行
    hideLoadingSpinner();
  });
```

# 正则表达式增强

## 命名捕获组

```js
const regex = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/;
const match = "2024-03-15".match(regex);
console.log(match.groups); // { year: '2024', month: '03', day: '15' }
```

## 后向断言

```js
// 正向后向断言
const regex1 = /(?<=\$)\d+/; // 匹配 $ 后面的数字
"Price: $123".match(regex1); // ['123']

// 负向后向断言
const regex2 = /(?<!\$)\d+/; // 匹配不在 $ 后面的数字
"Price: $123 and 456 items".match(regex2); // ['456']
```

## dotAll 模式 (s 标志)：

```js
const regex = /hello.world/s; // s 标志使 . 匹配换行符
const text = "hello\nworld";
regex.test(text); // true
```

## Unicode 属性转义

```js
const regex = /\p{Script=Greek}/u; // 匹配希腊字母
regex.test("π"); // true
```
