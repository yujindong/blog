### 粗体(Bold)

在要加粗的文本前后各添加两个星号

| Markdown               | HTML                                | preview              |
| ---------------------- | ----------------------------------- | -------------------- |
| `你好，我被**加粗**了` | `你好，我被<strong>加粗</strong>了` | 你好，我被**加粗**了 |

### 斜体(Italic)

在要使用斜体，在文本前后各添加一个星号

| Markdown         | HTML                   | preview        |
| ---------------- | ---------------------- | -------------- |
| `你好，我*歪*了` | `你好，我<em>歪<em>了` | 你好，我*歪*了 |

### 又粗(Bold) 又斜(Italic)

| Markdown                   | HTML                                          | preview                  |
| -------------------------- | --------------------------------------------- | ------------------------ |
| `你好，我***又粗又斜***了` | `你好，我<strong><em>又粗又斜<em></strong>了` | 你好，我**_又粗又斜_**了 |
